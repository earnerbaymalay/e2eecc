## 🧬 The Mission
**E2EECC** is a high-security communication client built on a **Zero-Knowledge Architecture**. It ensures that no intermediary—not even the server—can access the decrypted content of messages.

## 🔐 Security Protocols
- **Asymmetric Encryption**: RSA-4096 / ECC for secure key exchange.
- **Symmetric Encryption**: AES-256-GCM for real-time payload protection.
- **Forward Secrecy**: Unique session keys to prevent retroactive decryption.
- **Zero-Storage Policy**: No message data is ever stored permanently in cleartext.
